# 🌀 Fibonacci Sequence Generator

A simple **C++ console program** that prints the Fibonacci sequence up to `n` terms.  
This project demonstrates iterative programming, variable updates, and loop control.

---

## 📂 Repository

**Repository Name:** `fibonacci-sequence-generator`  
**Author:** [ayeshayousaf2555](https://github.com/ayeshayousaf2555)

---

## 📌 Features
- Accepts user input for the number of terms (`n`).
- Prints the Fibonacci sequence starting from 0 and 1.
- Uses iterative logic with variable updates (`firstprev`, `secondprev`, `result`).
- Efficiently generates sequence without recursion.

---

## ⚙️ Implementation Details
- Input: number of terms (`n`).
- Variables:
  - `firstprev` → stores the previous Fibonacci number.
  - `secondprev` → stores the second previous Fibonacci number.
  - `result` → stores the current Fibonacci number.
- Logic: iterative loop updates values and prints sequence until `n` terms are reached.
